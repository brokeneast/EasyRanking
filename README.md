# Easy Ranking 簡易排行榜
簡易排行榜能夠一一揭曉每位玩家所獲的分數，並依照分數高低排序。呈現已橫向長條圖的方式，將依照當前最高分為基準進行比例調整。

### 操作方法
----
點選組別名稱就能觸發跑分動畫。

![image](https://github.com/brokeneast/EasyRanking/blob/main/Demo/EasyRanking.gif)


### 設定
----
RankingManager可設置動畫速度及每位玩家名稱、分數、顏色。
